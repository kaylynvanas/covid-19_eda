# Novel Coronavirus 2019 Exploratory Data Analysis
**Data Source:** https://www.kaggle.com/sudalairajkumar/novel-corona-virus-2019-dataset/tasks?taskId=508
## Description
Coronaviruses are a family of viruses, with COVID-19 being the latest coronavirus discovered after an outbreak in many countries around the world. The Novel Coronavirus 2019 data set used in this [report](https://github.com/kaylynvanas/covid-19_eda/blob/master/EDA.docx) is a record of confirmed Novel Coronavirus 2019 (COVID-19) cases in the world accumulated from 22 January 2020 to 20 February 2020.

The analysis focuses on:
- where the outbreak started;
- where it has spread to;
- which locations are affected the most; and
- the rate at which total confirmed cases are increasing.

## Run
To run:
- Download the [Jupyter Notebook](https://github.com/kaylynvanas/covid-19_eda/blob/master/Novel%20Coronavirus%202019.ipynb) and [Data Set](https://github.com/kaylynvanas/covid-19_eda/blob/master/time_series_covid_19_confirmed.csv) to a folder, along with all other files in the repository;
- Open your terminal or command prompt and change directory to the folder using `cd`;
- Enter `jupyter notebook`;
- Once Jupyter Notebook is open, click Cell -> Run All in the menu.
